# Learning Resources — The Eternal Tree

If you're reading this, you've already done something remarkable: you opened
the source code. Most people never look behind the curtain. You did.

These documents explain how and why this application was built. They're
written for someone who wants to understand — not just copy — the code.

## Start Here

1. [**Architecture Story**](./architecture-story.md) — Why we chose each
   technology, and how they fit together. Start here.
2. [**The Streaming Pipeline**](./streaming-pipeline.md) — How a textbook
   passage becomes a tree of knowledge cards, step by step.
3. [**The Card System**](./card-system.md) — How rarity works, why we
   chose a TCG metaphor, and the psychology behind it.

## Prerequisites

You'll get the most out of these if you know:
- Basic HTML/CSS/JavaScript
- What a function is and how to call one
- How websites work (browser sends request, server responds)

You do NOT need to know React, Next.js, or TypeScript. We'll explain
everything as we go.
